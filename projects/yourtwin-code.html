Overview

YourTwin: CODE is a comprehensive full-stack coding education platform that leverages AI-powered digital twin technology to provide personalized, intelligent programming instruction. The platform combines real-time code execution, behavioral analytics, and machine learning to create adaptive learning experiences for computer science students.

Each student has an AI-powered virtual counterpart (digital twin) that learns from their coding behavior, predicts performance, and provides personalized guidance through a "Shadow Twin" AI assistant. The platform supports both student learning and instructor oversight, with advanced proctoring features, integrity monitoring, and comprehensive analytics.

This project demonstrates expertise in full-stack development, AI/ML integration, educational technology, and system design—all developed with support from Claude Code Agentic AI for intelligent architecture and implementation decisions.

Student Experience Features

  • Interactive Code Editor: Monaco Editor with syntax highlighting for C, C++, Java, and Python
  • Real-time Code Execution: Judge0 API integration for instant code testing and feedback
  • Web-based Terminal: xterm.js-powered terminal with interactive shell capabilities
  • Shadow Twin AI Assistant: 5-level progressive hint system based on scaffolding theory
  • Personality-Adaptive AI: 6 different AI personalities (Mentor, Socratic, Peer, Coach, Patient Teacher, Adaptive)
  • Digital Twin Dashboard: Behavioral analytics, competency visualization, learning velocity tracking, AI dependency analysis
  • Progress Tracking: Activity history, time analytics, success rate visualization, personalized recommendations

Instructor Experience Features

  • Lab Session Management: Multi-section support with dynamic enrollment and activity assignment
  • Real-time Monitoring: Live student activity tracking with status indicators and activity feeds
  • Activity Builder: Create coding activities with custom test cases and difficulty levels
  • Advanced Proctoring: Activity monitoring, integrity scoring, transcription detection, behavioral analysis
  • Analytics Dashboard: Performance metrics, AI usage analytics, behavioral insights, plagiarism detection

AI & Machine Learning Capabilities

  • MindSpore ML Models: LSTM success prediction, VAE anomaly detection, DQN learning path optimization, GAT knowledge graph modeling
  • Dual AI Architecture: DeepSeek-R1 for validation, Mistral for guidance, Qwen2.5-coder for solutions, Google Gemini fallback
  • Intelligent Code Analysis: Static analysis, behavioral validation, plagiarism detection, background processing queue
  • Personality-Adaptive Responses: Contextual assistance based on student competency, time spent, and previous attempts

System Architecture

The platform implements a sophisticated microservices architecture with three independent layers:

  • Frontend Layer: React SPA with Monaco Editor, xterm.js terminal, real-time dashboards, and responsive design
  • Backend Layer: Node.js/Express API with JWT authentication, Socket.io real-time communication, and comprehensive business logic
  • ML Layer: MindSpore-based microservice for LSTM predictions, VAE anomaly detection, DQN optimization, and GAT knowledge graphs
  • External Services: Judge0 for code execution, MongoDB for persistent storage, Ollama/Gemini for LLM inference

Core Innovation: Digital Twin Technology

  • Mirror Twin: Real-time behavioral data collection tracking typing speed, pause patterns, error frequency, and session metrics
  • Shadow Twin: AI assistant with 5-level progressive hint system adapting to individual learning styles and competency levels
  • Predictive Modeling: ML-driven success prediction and at-risk student identification enabling proactive interventions
  • Personalized Learning: Adaptive difficulty, content recommendations, and learning path optimization based on individual patterns

Academic Integrity & Security Features

  • Behavioral Monitoring: Detects tab switches, idle time, suspicious patterns without invasive surveillance
  • Plagiarism Detection: Token-based code similarity analysis with multiple algorithms
  • Lockdown Mode: Disables AI assistance during high-stakes assessments
  • Copy-paste Prevention: External paste blocking in restricted activities
  • Transparent Flagging: Clear integrity scoring and activity monitoring indicators
