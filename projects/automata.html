AUTOMATA

An AI-powered smart classroom system that automates the full session lifecycle — from facial recognition attendance and Gemini-generated lectures, to hand-raise detection, oral recitation with AI evaluation, and real-time live quizzes.

Add a screenshot here

Overview

AUTOMATA is a classroom automation platform designed to reduce administrative overhead for teachers and improve student engagement. A teacher uploads a lecture document; the system autonomously generates a structured Bloom's Taxonomy lecture, reads it aloud with AI-powered TTS, tracks attendance via facial recognition, detects hand raises using computer vision, conducts AI-evaluated oral recitation, and delivers a 30-question live quiz — all within a single session.

The platform separates the teacher and student experience completely. Teachers manage and monitor the session from a dashboard; students join via a session code and interact from their own devices. The session progresses through a defined lifecycle:

CREATED → ATTENDANCE → LECTURE → RECITATION → QUIZ → COMPLETED

Key Features

  • Facial Recognition Attendance: The teacher's camera scans the room and automatically identifies enrolled students using face-api.js. Undetected students can be marked present via a manual override panel.
  • AI Lecture Generation: Uploaded documents (PDF, DOCX, TXT) are processed through Google Gemini 2.5 Flash, which generates a structured six-section lecture following Bloom's Taxonomy — from foundational recall to higher-order creation.
  • Gemini TTS Delivery: Each lecture section is pre-converted to speech via Gemini TTS and cached as audio files. The teacher controls playback with per-section navigation and speed controls; comprehension checks auto-pause every three sections.
  • Hand-Raise Detection: A two-pipeline system combines MediaPipe Pose (crowd-level gesture detection) with face-api.js (individual identification). Pose detects that someone raised a hand; face recognition identifies who — linked by a 20-second attribution window.
  • Recitation with AI Evaluation: The teacher selects a student and records their spoken answer via the Web Speech API. The transcript is sent to Gemini, which returns a scored evaluation (Excellent / Good / Satisfactory / Needs Improvement) and written feedback.
  • Real-Time Live Quiz: A 30-question MCQ quiz (5 questions per Bloom level) is broadcast to all connected students simultaneously via WebSockets. The teacher sees a live submission counter and a colour-coded results table as students complete it.
  • Dual-Role Authentication: Separate teacher and student accounts with independent session guards — teachers manage sessions, students enroll via code and register their face during sign-up.
