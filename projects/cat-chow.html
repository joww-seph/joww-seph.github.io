Projects CAT-chow

CAT-chow

A 2D pixelated side-scrolling platformer built entirely in Java where you play as Chao-chao, a cat battling through 7 stages of enemies, obstacles, and boss encounters. The game is complete with power-ups, enemy AI, and a save/load system.

Overview

CAT-chow is a 2D pixelated side-scrolling platformer developed entirely in Java using Java Swing and AWT. You play as Chao-chao, a lost cat abandoned by its owner when their house was demolished. As Chao-chao, you pass through 7 stages, each with its own tile map, music, enemy layout, and collectibles. Every system used within the game was built from scratch using only the Java standard library - the tile renderer, collision detection, entity AI, camera, input handler, sound engine, and save/load.

Key Features

  • 7 unique stages with individual maps and music tracks
  • 4 power-ups - speed boost, double jump, air slash, and invisibility
  • 4 enemy types with distinct AI behaviors and stats
  • Stage 5 airplane mode - full mechanic swap with auto-scrolling and new controls
  • Cutscenes between stage transitions and a full ending sequence
  • Save/load system to persist player progression
  • 60 FPS game loop with game state machine

Stages

Stage 1 - Subdivision

No enemies. Introduces movement, power-ups, and the exit mechanic.

Stage 2 - Rooftops

Introduces Milky and Garfield with patrol and pursuit AI. All four power-up types appear for the first time.

Stage 3 - Bridge

Introduces the Hawk, a fast aerial enemy that tracks the player's vertical and horizontal position.

Stage 4 - City

Vehicles - buses, trucks, vans, and cars - fill the world as platforms and obstacles, alongside the heaviest enemy density in the game.

Stage 5 - Air

The player transforms into an airplane. The screen auto-scrolls left and controls switch to W / S only. Hawks swarm throughout.

Stage 6 - Power-Up Gauntlet

Dense power-up placement offsets the highest total enemy count of any stage.

Stage 7 - Apartment

Introduces the Human - the most dangerous enemy with the highest attack damage. Set in an indoor environment filled with furniture obstacles. Completing this stage rolls the ending cutscene.

Enemies

  • Milky - Fast, fragile patrol enemy (1 HP). Reverses on hit.
  • Garfield - Tankier pursuer (3 HP, 2 ATK). Chases within range.
  • Hawk - Aerial tracker (99 HP, speed 12). Adjusts to the player's position every 120 frames. Main threat in Stage 5.
  • Human - Slow but hits hard (99 HP, 3 ATK). Stage 7 exclusive.

What I Learned

CAT-chow was my first complete game built without any engine. Every system had to be written by hand, which forced me to understand game development at a low level - from maintaining a stable 60 FPS game loop to implementing tile-based collision detection and managing game state across different screens.

The most creative challenge was Stage 5's airplane mode - swapping the player's sprite, physics, collision box, and input scheme mid-game without breaking the existing loop. It taught me to keep state and rendering cleanly separated. Looking back, the biggest constraint was Java Swing itself, a UI toolkit, not a game framework. Thus, I plan to rebuild this in a proper engine like Godot as the next step.
