MemeRise

Transform study notes into memorable mnemonic memes. AI-powered mnemonic generation paired with 30+ meme templates creates a dual-sensory learning experience that makes memorization engaging and fun.

Overview

MemeRise is an AI-powered web application designed to transform study notes and lists into memorable mnemonic memes. The application automatically generates clever, memorable sentences from the first letters of your items (mnemonic devices) and pairs them with popular meme templates or AI-generated images, making learning more engaging and fun.

This project demonstrates full-stack web development skills with integration of cutting-edge AI technologies to solve a real educational problem: helping students memorize complex lists and concepts through humor and memorable imagery.

Problem & Solution

Problem: Students often struggle to memorize lists and concepts (e.g., planets, historical dates, scientific processes). Traditional mnemonics require manual creation and lack visual engagement.

Solution: MemeRise automates mnemonic generation using AI and combines it with visual meme templates, creating a dual-sensory learning experience that leverages humor and memorable imagery.

Key Features

  • AI-Powered Mnemonic Generation: Google Gemini API automatically generates clever, funny mnemonics from user input lists
  • 30+ Meme Template Library: International classics (Drake, Distracted Boyfriend, Expanding Brain) plus 20+ Filipino meme favorites (Harry Roque, Mimiyuhhhhh, Si OA at Nonchalant)
  • AI Image Generation: Pollinations AI API creates custom illustrations for the default template
  • Custom Meme Creator: Dedicated page with drag-and-drop image upload, canvas-based text overlay, and PNG export
  • Canvas-Based Rendering: Custom drawing functions for each template with optimal text placement and readability
  • Browser-Based Export: Instant PNG download without server requirements

Technical Architecture

Frontend Stack: HTML5 semantic markup, Vanilla JavaScript (ES6+), Tailwind CSS for modern responsive styling, and HTML5 Canvas API for meme generation and image manipulation.

Backend/APIs: Google Gemini API for mnemonic generation, Pollinations AI API for custom image generation. No backend server—fully client-side application with API integration.

Project Structure: Main interface (index.html/index.js), Custom meme creator (custom.html/custom.js), API configuration (config.js), and Assets folder with templates and icons.

How It Works

  1. User enters a title (e.g., "Planets of the Solar System")
  2. User inputs items to memorize (comma or newline separated)
  3. Application sends items to Google Gemini API
  4. AI generates a memorable mnemonic sentence
  5. User selects a meme template from 30+ options
  6. Canvas API draws the mnemonic text onto the selected template image
  7. User downloads the generated meme as PNG

Challenges & Solutions

Mnemonic Quality Consistency: Fine-tuned Gemini API prompts with examples and specific instructions for humor and memorability to ensure generated mnemonics are both funny and actually memorable.

Template Text Placement: Created individual canvas drawing functions for each template with custom text layout logic to ensure optimal visual appeal for different meme formats.

Image Generation Integration: Integrated Pollinations AI API to generate context-aware illustrations on-the-fly, providing custom images without requiring user-provided assets.

File Upload Handling: Implemented FileReader API with size/type validation and visual feedback for drag-drop and click-upload functionality in the custom meme creator.

Skills Demonstrated

Frontend Development: HTML5 Canvas API, Vanilla JavaScript (ES6+), responsive CSS design with Tailwind, event handling, DOM manipulation

API Integration: Async/await REST API calls, error handling, working with external services (Gemini, Pollinations), configuration management

Software Engineering: Problem decomposition, modular JavaScript architecture, user-centered design, testing and debugging

Design & UX: Visual feedback systems, responsive design, intuitive workflows, accessibility considerations

What I Learned

Creating MemeRise provided valuable experience in AI/ML integration, Canvas API manipulation, asynchronous programming with API calls, and UX design for educational applications. I deepened my understanding of how humor and creativity enhance memory retention, and the importance of designing interfaces that make learning enjoyable.
